# Fixel Fonts

Place the Fixel font files here. Download them from the official MacPaw release:

**GitHub:** https://github.com/MacPaw/fixel/releases

Files needed:
- `FixelDisplay-Regular.ttf`
- `FixelDisplay-SemiBold.ttf`
- `FixelDisplay-Bold.ttf`
- `FixelText-Regular.ttf`
- `FixelText-SemiBold.ttf`
- `FixelText-Bold.ttf`

After placing the files here, run:
```
flutter pub get
flutter run
```
