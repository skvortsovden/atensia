Атенція — це безкоштовний додаток для щоденних записів про твій настрій, самопочуття та дозвілля. Він допоможе тобі помічати зміни та краще розуміти себе. Усі дані зберігаються лише на твоєму пристрої. Додаток не вимагає реєстрації, не збирає жодних даних та не використовує інтернет.

Функціонал додатку включає:

  • Щоденні записи про самопочуття та дозвілля
  • Календар для оновлення записів за минувші дні
  • Звіт для розуміння змін за певний період
  • Нагадування, щоб не забувати робити записи
